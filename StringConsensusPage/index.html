String Consensus Python Package

About the Package

The field of computational biology is an exciting and rising interdisciplinary field. It encompasses practices from the broader fields of Machine Learning, Artificial Intelligence, and Optimization

Classical problems in Computational Biology are String Consensus problems, involved in practices like Genetic Sequencing. These problems involve the appraisal of sequences of character in order of determining a single string of characters that is either the closest to all the sequences or the furthest. These problems are not straighforwards, so a variety of algorithms and heuristics must be applied in order of obtaining an optimal solution.

The objective of this project is to develop and offer a package for the Python programming language that enables users to define and model different instances of problems to get the best solution for both Closest String Problems and Far From Most String Problems

This project is proposed and supervised by Dr. Giovanni Stracquadanio
